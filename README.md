# Compilacion-sensores
Esta carpeta viene compuesta por 2 codigos, un codigo para emisor y un codigo para receptor
El codigo del emisor esta adaptado a un arduino nano y un sensor de flujo de agua y un NRF-l2401.
El codigo del receptor esta adaptado a un arduino MEGA con un sensor de interruptor magnetico MC-38, 
un sensor de movimiento piroelectrico HCSR04, un sensor de corriente ACS712, y el modulo NRF-l2401
